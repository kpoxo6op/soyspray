# Operator documentation

This folder is the source for the Kong bank lab guide published with MkDocs.
GitHub renders every Markdown page directly, while [`../mkdocs.yml`](../mkdocs.yml)
controls the hosted site navigation.

## Suggested reading order

1. [`getting-started.md`](getting-started.md): prerequisites and first run
2. [`architecture.md`](architecture.md): components and traffic flow
3. [`api-catalog.md`](api-catalog.md): sample APIs and policy profiles
4. [`operations.md`](operations.md): normal operating tasks
5. [`runbooks/`](runbooks/): step-by-step procedures

[`repository-map.md`](repository-map.md) links documentation concepts to the
corresponding code folders.
