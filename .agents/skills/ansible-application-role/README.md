# Ansible application role skill

This project-owned skill covers Ansible roles that create and remove Argo CD
applications and related Kubernetes resources.

- [`SKILL.md`](SKILL.md) contains the full workflow and review checklist.
- [`agents/openai.yaml`](agents/) contains the short name and default prompt
  shown by compatible agent tools.

Use this skill for defaults, task structure, enable/disable behaviour,
idempotency, check mode, and safe GitOps deployment.
