# Demo lab experience skill

This project-owned skill treats a technical lab as a small product for both
operators and demo visitors.

- [`SKILL.md`](SKILL.md) covers setup, power controls, sample applications,
  browser behaviour, evidence, reset, and shutdown.
- [`agents/openai.yaml`](agents/) contains the agent-facing name and prompt.

Use this skill when a lab works technically but its purpose or operating path
is hard to understand.
