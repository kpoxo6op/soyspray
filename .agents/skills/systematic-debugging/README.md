# Systematic debugging skill

This vendored skill requires root-cause evidence before a fix is proposed.

- [`SKILL.md`](SKILL.md) contains the four-phase debugging workflow.
- [`root-cause-tracing.md`](root-cause-tracing.md) traces bad state back to its
  source.
- [`defense-in-depth.md`](defense-in-depth.md) adds validation after the root
  cause is known.
- [`condition-based-waiting.md`](condition-based-waiting.md) replaces arbitrary
  sleeps with observable conditions.
