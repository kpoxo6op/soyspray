# Test-driven development skill

This vendored skill uses a failing test, the smallest implementation, and a
fresh passing run for each feature or bug fix.

- [`SKILL.md`](SKILL.md) contains the red-green-refactor workflow.
- [`testing-anti-patterns.md`](testing-anti-patterns.md) explains tests that can
  give false confidence.
