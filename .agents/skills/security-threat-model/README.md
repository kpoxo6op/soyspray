# Security threat model skill

This vendored skill builds repository-grounded threat models with assets, trust
boundaries, abuse paths, and mitigations.

- [`SKILL.md`](SKILL.md) contains the workflow.
- [`references/`](references/) contains the report template and control catalog.
- [`agents/`](agents/) contains agent-interface metadata.

Upstream version and licence details are recorded in
[`../SOURCES.md`](../SOURCES.md) and [`LICENSE.txt`](LICENSE.txt).
