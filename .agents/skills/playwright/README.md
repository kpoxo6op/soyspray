# Playwright skill

This vendored skill guides real-browser automation from the terminal.

- [`SKILL.md`](SKILL.md) contains the operating rules.
- [`references/`](references/) explains the command-line interface and common
  workflows.
- [`scripts/`](scripts/) contains the repository wrapper.
- [`assets/`](assets/) contains display images.
- [`agents/`](agents/) contains agent-interface metadata.

Upstream version and licence details are recorded in
[`../SOURCES.md`](../SOURCES.md) and [`LICENSE.txt`](LICENSE.txt).
