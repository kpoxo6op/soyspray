# Agent guidance

This folder keeps reusable instructions for automated coding and review work.
The content is versioned with the repository so the same engineering rules can
be applied again in later features.

## Start here

- [`skills/`](skills/) contains focused engineering playbooks.
- [`skills/SOURCES.md`](skills/SOURCES.md) records which skills are maintained
  here and which are copied from upstream projects.

Project-owned skills may be changed with the repository. Vendored skills should
only be updated from their recorded upstream source.
