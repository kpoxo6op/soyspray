# Mosquitto

Mosquitto is an open source MQTT message broker that implements the MQTT protocol versions 5.0, 3.1.1 and 3.1.

## Features

- MQTT protocol support (versions 5.0, 3.1.1, 3.1)
- WebSockets support for browser-based clients
- Message persistence
- Authentication and authorization support

## Integration

Mosquitto serves as the MQTT broker for home automation systems, enabling communication between IoT devices, sensors, and automation platforms through the MQTT publish/subscribe messaging pattern.
