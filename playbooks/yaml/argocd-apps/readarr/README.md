# readarr

## Purpose
User-friendly interface for searching and tracking books, which uses Jackett to find torrents and sends them to qBittorrent.

## Minimal Validation (Before Moving On)
• Web UI is reachable.
• Jackett indexer is added under 'Indexers'.
• qBittorrent is configured under 'Download Clients'.
• Searching for a book triggers a torrent download in qBittorrent.

## Ladder Progression
**This is the final step** for a basic book-searchable torrent chain. Enjoy!
